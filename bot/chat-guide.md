# Chat Guide

## View Your Chats

The **Chats** page displays all conversations you are part of, sorted by
the most recently updated.

Available actions: - Change the sorting order by clicking the **Arrows**
button. - Filter chats by clicking the **Funnel** button and selecting a
chat type.

## Create a New Chat

### Start a Direct Chat

1.  Go to the **People** page.
2.  Use the search bar to find the desired user.
3.  Click **Open Chat** on the user card.

### Create a Group Chat

1.  Go to the **Chats** page.
2.  Click the **+** button at the top.
3.  Select the chat type:
    -   **Direct Chat** → redirects to the **People** page.
    -   **Group Chat** → redirects to **Chats / New**.
4.  To create a group chat:
    -   Enter a group name.
    -   Use the **Users** panel to search for, add, or remove members.
    -   Click **Create Group Chat**.

Group settings can be modified later if needed.

## Manage Chat Settings

Only **group chat admins** can manage chat settings, including editing
group details, managing members, and deleting chats.

Available admin actions: - Edit group name - Manage users - Delete chat

### Admin Rules

-   In the **Users** settings of the chat, admins have a **Crown** icon
    near their names.
-   A group chat will always have at least one admin.
-   Only admins can make other users admins.
-   If a chat has no users, the chat is automatically deleted.

### Edit Group Name

1.  Open the group chat.
2.  Click the **3-dot menu** in the chat header.
3.  Select **Settings**.
4.  Click the **Pencil** icon to enable editing.
5.  Update the group name.
6.  Click **Check** to save or **X** to cancel.

### Manage Users

1.  Open the group chat.
2.  Click the **3-dot menu** in the chat header.
3.  Open the **Users** panel.

Here you can: - View all participants.

As an admin, you can also: - Remove a user using the **X** button. -
Grant admin permissions using the **Crown** button. - Remove admin
permissions using the **Crossed Crown** button.

#### Add Users

1.  Click **Add Users**.
2.  Search for users in the panel.
3.  For each result:
    -   Click **Check** to add the user.
    -   Click **X** to remove the user if already added.

### Delete a Chat

1.  Open the chat.
2.  Click the **3-dot menu** in the chat header.
3.  Select **Advanced Settings**.
4.  Expand the section if needed.
5.  Click **Delete Chat**.
6.  Confirm or cancel the deletion.

## Send and Receive Messages

Messages can be viewed and sent from the **Chat** page.

### View Messages

-   Messages are displayed in chronological order, with the newest
    messages shown at the bottom.
-   New messages appear instantly in an open chat.
-   If the chat is closed, new messages are shown as notifications at
    the top of the page.
-   Load older messages by scrolling upward.

### Send Messages

1.  Open a chat.
2.  Type your message in the **Message Editor**.
3.  The editor supports **rich text formatting** through a WYSIWYG
    interface.
4.  Send your message.

### Manage Sent Messages

After sending a message, you can: - **Edit** the message within **10
minutes** of sending. - **Delete** the message.

### Reply to Messages

-   Reply to a specific message to create a reference to the original
    message.
